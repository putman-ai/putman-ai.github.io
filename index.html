Open Source Repos

Starting blocks for some of my internal research and projects.

HRP Optimization

Hierarchical Risk Parity (HRP) is a portfolio optimization strategy that uses machine learning to assess asset correlation structures, resulting in diversified and risk-efficient portfolios.

Black-Scholes

Black-Scholes is a model used for pricing options by considering factors like volatility, time, and risk-free rate, assuming markets are efficient and volatility is constant. This is a very basic example.

HC for Feature Selection

This is an early stage prototype for feature selection using hierachical clustering. It will be tested in the Numerai comp along with Boruta-Shape groupings etc.
